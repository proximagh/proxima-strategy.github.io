# proxima-strategy.github.io

Public website for Proxima Strategy Ltd, served via GitHub Pages.

## Setup

No build step required. Push changes to `main` and GitHub Pages will serve `index.html` automatically.

**URL:** https://proxima-strategy.github.io

## To enable GitHub Pages

1. Repo Settings → Pages
2. Source: Deploy from branch → `main` → `/ (root)`
3. Save

## Structure

```
/
├── index.html       # Single-page site
├── README.md
└── assets/          # Add images, icons here
```
